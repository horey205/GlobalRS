# 🛰️ 글로벌 위성탐사 분석지도: 학생 실습 가이드

이 프로젝트는 Microsoft Planetary Computer의 위성 데이터를 활용하여 환경을 분석하는 웹 어플리케이션입니다.

## 🚀 시작하기 (설치 및 실행)

각자의 컴퓨터에서 아래 명령어를 순서대로 입력하세요.

1. **패키지 설치**:
   ```bash
   npm install
   ```

2. **개발 서버 실행**:
   ```bash
   npm run dev
   ```
   실행 후 터미널에 뜨는 `http://localhost:5173` 주소를 브라우저로 여세요.

## 🧪 도전 과제 (직접 고쳐보기!)

`src/App.jsx` 파일을 열어서 아래 내용들을 수정해 보세요.

### 1. 색상 테마 바꾸기
- 지도의 NDVI(식생) 색상이 맘에 안 드나요? `getTileUrl` 함수에서 `colormap_name`을 찾아보세요.
- `rdylgn` (빨강-노랑-초록) 대용으로 `viridis`, `magma`, `greens` 등을 넣어보세요.

### 2. 나만의 분석 지수 만들기
- `runAIAnalysis` 함수에는 NDVI와 NBR 식(`(B08-B04)/(B08+B04)`)이 있습니다.
- 새로운 지수를 찾아보거나, 수치를 조절해서 분석 지능을 바꿔보세요.

### 3. UI 디자인 개성 있게 바꾸기
- `App.jsx` 하단의 JSX 영역(HTML 부분)에서 버튼의 글자나 색상(`background`)을 바꿔보세요.
- 나만의 위성 분석 서비스 이름을 지어보세요!

## ⚠️ 주의사항
- 위성 탐색 시 날짜 범위를 너무 길게 잡으면 탐색 시간이 오래 걸릴 수 있습니다.
- 구름이 너무 많은 날은 분석 결과가 부정확할 수 있으니 클라우드 커버 수치를 확인하세요.
